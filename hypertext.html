HTML and CSS

HTML (abbreviation for HyperText Markup Language) is a markup language for hypertext.CSS (Cascading Style Sheets) is a language that describes the presentation style of a structured document such as font, text size and color. The CSS technology is a way to customize the document with regard to computer type, screen resolution, color depth and installed fonts.
